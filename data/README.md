# Dataset

Este proyecto utiliza el dataset **Smart Meters in London** de Jean-Claude Cote,
disponible públicamente en Kaggle bajo licencia CC0 (Public Domain).

## Cómo obtener el dataset

1. Crear una cuenta gratuita en [Kaggle](https://www.kaggle.com)
2. Ir a: https://www.kaggle.com/datasets/jeanmidev/smart-meters-in-london
3. Descargar el archivo `halfhourly_dataset.zip`
4. Subir el zip a Google Drive
5. En el notebook, montar Drive y apuntar la ruta a ese archivo

## Características

| Atributo | Detalle |
|---|---|
| Hogares | 5.567 |
| Período | Diciembre 2011 – Febrero 2014 |
| Frecuencia | Cada 30 minutos |
| Filas aprox. | 1,5 millones por archivo |
| Formato | CSV comprimido (.zip) |
| Licencia | CC0 Public Domain |
